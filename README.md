## 🐧 Linux Experience - Guia Prático

Anotações e guias práticos do curso **Linux Experience** na plataforma DIO. Foco em Linux/Ubuntu para DevOps.

## O que você encontra aqui

- Instalação e configuração de Ubuntu Server 24.04
- Navegação e comandos de linha de comando
- Gerenciamento de permissões (chmod, chown, sudo)
- Usuários e grupos
- Administração básica do sistema
- Preparação do ambiente para DevOps

## 📚 Estrutura

```
└── linux-ubuntu-24-04/    # Guias de instalação e prática
```

## 🎯 Para quem é

Ideal para profissionais migrando da infraestrutura tradicional para DevOps/Cloud e precisam dominar Linux na prática.

## 💡 Como usar

Os arquivos estão organizados por tópico. Comece por ordem e pratique cada comando em um ambiente de teste.

## 🔧 Pré-requisitos

- VirtualBox ou hipervisor disponível
- Imagem ISO do Ubuntu Server 24.04
- ~20 GB de espaço em disco

Erros? Sugestões? Fique à vontade para abrir issues ou PRs.
